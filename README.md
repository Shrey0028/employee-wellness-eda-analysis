# Employee Wellness Analysis using Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project analyzes employee wellness and mental health using Exploratory Data Analysis (EDA). The objective is to identify key workplace factors influencing employees' mental health treatment and provide actionable business recommendations.

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook / Google Colab

## 📊 Project Workflow

- Data Collection
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insights
- Recommendations

## 📈 Key Findings

- Family history is strongly associated with mental health treatment.
- Work interference significantly impacts treatment-seeking behavior.
- Employer support and wellness programs encourage employees to seek professional help.
- Supervisor and coworker support contribute to a healthier workplace environment.
- Remote work influences employee well-being differently depending on organizational support.
